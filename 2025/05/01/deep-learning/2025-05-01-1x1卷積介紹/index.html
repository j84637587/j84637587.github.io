1x1 卷積層介紹

發表於 2025-05-01 更新於 2026-03-10

什麼是 1x1 卷積層？

1x1 卷積層是一種特殊的卷積操作，其中卷積核的尺寸為 1x1。這意味著每次卷積操作只會影響輸入特徵圖中的單個像素，但會跨通道進行計算。

1x1 卷積層的用途

1x1 卷積層在深度學習中有以下幾個主要用途：

  1. 降維或升維：通過調整輸出通道數，1x1 卷積層可以用於壓縮或擴展特徵圖的通道數，從而減少計算量或增加模型表達能力。
  2. 非線性映射：1x1 卷積層可以引入非線性激活函數（如 ReLU），從而增加模型的非線性表達能力。
  3. 特徵融合：通過跨通道的加權求和，1x1 卷積層可以實現不同通道特徵的融合。

1x1 卷積層的數學表示

假設輸入特徵圖的尺寸為 ( H \times W \times C_{in} )，1x1 卷積層的輸出特徵圖尺寸為 ( H \times W \times C_{out} )。其計算公式為：

[
Y_{h,w,c} = \sum_{i=1}^{C_{in}} X_{h,w,i} \cdot W_{i,c} + b_c
]

其中：

  • Xh,w,iX_{h,w,i}Xh,w,i​ 是輸入特徵圖在位置 (h,w)(h, w)(h,w) 和第 iii 通道的值。
  • ( W_{i,c} ) 是 1x1 卷積核的權重。
  • ( b_c ) 是偏置項。
  • ( Y_{h,w,c} ) 是輸出特徵圖在位置 ( (h, w) ) 和通道 ( c ) 的值。

1x1 卷積層的優勢

  • 計算效率高：相比於更大的卷積核，1x1 卷積層的計算量更小。
  • 靈活性強：可以靈活調整通道數，適用於各種網絡結構。
  • 應用廣泛：在 GoogLeNet（Inception 模型）和 ResNet 等經典網絡中廣泛使用。

結論

1x1 卷積層是一種簡單但功能強大的卷積操作，能夠有效地提升模型的表達能力，同時降低計算成本。在設計卷積神經網絡時，合理使用 1x1 卷積層可以帶來顯著的性能提升。
